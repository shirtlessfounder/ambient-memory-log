# Ambient Memory Log

Recall-first ambient memory log MVP.

## Setup

1. Install `ffmpeg` on macOS.
2. Install project dependencies with `uv sync`.
3. Copy `.env.example` to `.env` and fill in the required values.

## Commands

- `uv run ambient-memory --help`
- `uv run ambient-memory list-devices`
- `uv run ambient-memory agent --help`
- `uv run ambient-memory worker --help`
- `uv run ambient-memory api --help`
- `uv run ambient-memory enroll --help`
